Desafíos Multitemáticos de Ciberseguridad

Una plataforma interactiva para el aprendizaje y entrenamiento en ciberseguridad.

Objetivos del Proyecto

El objetivo principal es diseñar, desarrollar e implementar un aplicativo web que proporcione un entorno de aprendizaje práctico y dinámico para el fortalecimiento de habilidades en ciberseguridad. Esto se logrará mediante:

  • La oferta de desafíos variados
  • Un sistema de puntuaciones competitivas
  • La generación de reportes detallados

Roles del Aplicativo Web

Administrador

  • Creación y gestión de cuentas de usuarios (docentes)
  • Configuración de cursos disponibles
  • Parametrización de trivias
  • Asignación de permisos y roles
  • Seguimiento y gestión de puntuaciones
  • Generación de reportes sobre el desempeño general

Docente

  • Construcción de trivias
  • Parametrización de fechas de aplicación
  • Asignación de trivias
  • Generación de reportes detallados

Estudiante

  • Visualización de trivias asignados
  • Realización de trivias
  • Visualización de puntuaciones
  • Ingreso de nombre para registro

Tecnologías Utilizadas

El desarrollo del aplicativo web se llevará a cabo utilizando las siguientes tecnologías:

Backend

  • Lenguaje de programación: Java
  • Framework: Spring Boot
  • ORM: Hibernate
  • Base de datos: PostgreSQL

Frontend

  • Framework: Angular
  • Gestión de versiones: Git
  • Servidor de aplicaciones: Apache Tomcat o Spring Boot incorporado
